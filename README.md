# 🎓 MBTI 기반 학과 추천 웹페이지

MBTI 성격 유형 테스트를 통해 사용자에게 가장 적합한 대학 학과를 추천해주는 인터랙티브 웹사이트입니다.

## ✨ 주요 기능

- **MBTI 성격 유형 테스트**: 12개의 질문으로 구성된 정확한 MBTI 진단
- **맞춤형 학과 추천**: 16가지 MBTI 유형별 최적화된 학과 추천
- **상세한 설명**: 각 학과의 특징, 매칭도, 주요 기능 소개
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 완벽 지원
- **현대적 UI/UX**: 직관적이고 아름다운 사용자 인터페이스

## 🚀 사용 방법

1. 웹페이지 접속
2. "테스트 시작하기" 버튼 클릭
3. 12개 질문에 솔직하게 답변
4. MBTI 결과 및 추천 학과 확인
5. 결과 공유 또는 재테스트 가능

## 📊 테스트 구성

총 12개의 질문으로 4가지 MBTI 지표를 측정:
- **E (외향) vs I (내향)**: 에너지 방향성
- **S (감각) vs N (직관)**: 정보 수집 방식
- **T (사고) vs F (감정)**: 의사결정 방식
- **J (판단) vs P (인식)**: 생활 양식
